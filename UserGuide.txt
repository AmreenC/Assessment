This application is used to Sort the Given list of names, order set by first by Surname/LastName. 
It prints all the sorted names to console and writes the sorted list of names to a file called sorted-names-list.txt.
Also UnitTest are provided.

To Run the Application at Command Line, follow below given command:
1.Go to name-sorter directory and then 
  dotnet run <ArgPathOfFileToBeSorted>
e.g. dotnet run ./unsorted-names-list.txt 
OR
2.Go to name-sorter directory and then run exe file
  Program <ArgPathOfFileToBeSorted>
e.g Program ./unsorted-names-list.txt
